## 方針

このレベルでは、講義ノートの「挿入で直す4箇所」をCのポインタで実装します。

## ポイント

`p` の次に `x` を入れるとき、もともと `p` の次にあったノードを `q` とします。

```c
struct DNode *q = p->next;
x->prev = p;
x->next = q;
p->next = x;
if (q != NULL) {
    q->prev = x;
}
```

先頭挿入では `p` がありません。この場合、古い `head` が `q` になります。

## 注意点

更新順を雑にすると、もとの `p->next` を失うことがあります。先に `q = p->next` と保存してから書き換えます。
