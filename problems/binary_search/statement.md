# binary_search

昇順に並んだ配列 `A` から値 `x` を二分探索してください。

`x` が見つかった場合は位置を `0-indexed` で出力します。見つからない場合は `-1` を出力します。

## 入力

```text
n x
a[0] a[1] ... a[n-1]
```

## 出力

```text
位置 または -1
```

## 制約

- `1 <= n <= 1000`
- `A` は昇順にソート済み
- 配列の要素と `x` は整数

## 入力例

```text
5 4
1 2 3 4 5
```

## 出力例

```text
3
```
